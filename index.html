gnx86

gnx86 is not x86. But it is a free (as in freedom) computer, or at least a seed for one.

Download this project as a .zip file Download this project as a tar.gz file

gnx86 is a free computer. Just not a physical one.

The goal is to make a simple, implementable, yet readable computer architecture, based on a register machine with interrupts. Free computers are necessary for GNUsers, since the GNU freedoms must include hardware to make any sense.

gnx86 has nothing to do with GNU, except that it considers it a friend.
